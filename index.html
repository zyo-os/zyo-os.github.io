Por que escolher o ZYO-OS?

Construído sobre ombros de gigantes, refinado para o usuário final.

Privacidade Total

Sem telemetria, sem coleta de dados e sem IA acoplada ao sistema. Seus dados permanecem no seu computador.

Simplicidade

Interface pensada para quem não tem afinidade com tecnologia. Ideal para crianças e idosos.

Base Debian Sólida

Estabilidade inabalável do Debian Bookworm. Milhares de pacotes disponíveis e segurança de nível empresarial.

Faça Você Mesmo

Não apenas um sistema, mas uma escola. Nossa documentação ensina você a construir o sistema do zero.

Como Funciona a Criação?

O ZYO-OS não é apenas um sistema, é um método. Veja como você construirá seu OS.

01

Sistema Base

Começamos com o debootstrap para criar um núcleo Debian puro e mínimo, sem bloatware.

02

Interface

Instalamos o ambiente gráfico (GNOME ou XFCE) e configuramos drivers de vídeo e áudio.

03

Personalização

Aplicamos temas, ícones, papéis de parede e configurações de segurança (Firewall/DNS).

04

Geração da ISO

Compactamos tudo em um arquivo .iso bootável pronto para instalar em qualquer PC.

Sobre o Projeto ZYO

O ZYO-OS nasceu da necessidade de simplificar o Linux para usuários não técnicos, mantendo a robustez do Debian.

Nossa filosofia é baseada em três pilares:

  • Transparência: Código 100% aberto e auditável.
  • Educação: Ensinar como o sistema funciona, não apenas entregar uma caixa preta.
  • Privacidade: Zero coleta de dados. O computador é seu.

Perguntas Frequentes

Preciso saber programar?

Não! O guia é passo-a-passo. Apenas familiaridade básica com terminal ajuda, mas explicamos cada comando.

Funciona em computadores antigos?

Sim! A versão com XFCE é extremamente leve e roda bem em máquinas com 2GB de RAM ou menos.

É seguro para operações bancárias?

Sim. Baseado no Debian Stable, o ZYO-OS herda toda a segurança de nível empresarial do Debian, somado a configurações extras de DNS e Firewall.

Pronto para assumir o controle?

Você pode baixar a imagem ISO pronta para uso ou aprender a construir seu próprio sistema operacional passo a passo.
